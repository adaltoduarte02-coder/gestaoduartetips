GESTAO DUARTE TIPS PRE-LIVE - DEPLOY

Opcao rapida para site estatico:
1. Acesse https://app.netlify.com/drop
2. Arraste a pasta gestaoduartetips-prelive-pronto ou o arquivo gestaoduartetips-prelive.zip
3. Depois do deploy, clique em Site configuration > Change site name
4. Use um nome como gestaoduartetips

O link gratis deve ficar parecido com:
https://gestaoduartetips.netlify.app

Dominio proprio:
1. Compre ou use um dominio como gestaoduartetips.com.br
2. Na Netlify, va em Domain management
3. Clique em Add a domain
4. Aponte o DNS conforme a Netlify mostrar

Arquivos importantes:
- index.html: app principal
- app.js: logica de gestao, analises pre-live e dashboard
- style.css: visual principal do app
- assets/mentor.png: imagem da capa

Observacao:
Esta versao funciona como site estatico.
Para dados reais de jogos, rankings e historicos, plugue uma API de futebol no proximo passo.
