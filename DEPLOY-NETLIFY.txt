GESTAO DUARTE TIPS PRE-LIVE COM API - DEPLOY

Publicar pelo GitHub/Netlify:
1. Suba todos os arquivos e pastas deste pacote no GitHub.
2. No Netlify, o projeto deve publicar pela branch main.
3. Configure a variavel secreta API_FOOTBALL_KEY no Netlify.
4. Va em Deploys > Trigger deploy > Deploy project.

Esta versao usa backend do Netlify:
- netlify/functions/football-fixtures.js busca os jogos reais do dia.
- netlify/functions/football-analysis.js monta a analise quando clicar no jogo.

O link gratis deve ficar parecido com:
https://gestaoduartetips.netlify.app

Dominio proprio:
1. Compre ou use um dominio como gestaoduartetips.com.br
2. Na Netlify, va em Domain management
3. Clique em Add a domain
4. Aponte o DNS conforme a Netlify mostrar

Arquivos importantes:
- index.html: app principal
- app.js: logica de gestao, analises pre-live e dashboard
- style.css: visual principal do app
- netlify/functions: integracao segura com API-Football
- API-FOOTBALL-CONFIG.txt: passo a passo da variavel secreta
- assets/mentor.png: imagem da capa

Observacao:
Sem API_FOOTBALL_KEY no Netlify, a aba Analises mostra aviso e nao inventa jogos.
